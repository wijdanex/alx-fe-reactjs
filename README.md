# GitHub User Search (github-user-search)


A small React (Vite) app to search GitHub users via the GitHub REST API.


## Quick Start
1. Install dependencies: `npm install`
2. Create a `.env` file from `.env.example` if you want to add a GitHub API key (optional).
3. Run the dev server: `npm run dev`


This repository is part of the ALX "Working with APIs" project.
